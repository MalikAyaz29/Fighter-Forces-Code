# 🎖️ Fighter Forces Academy Mianwali (FFAMW)
### *Elite Defense Forces Preparation*

Prepare for a career in the Pakistan Armed Forces with the most comprehensive online preparation platform. **FFAMW** is designed to help candidates excel in Pakistan Army, Navy, and Air Force recruitment tests through interactive tools and up-to-date guidance.

---

## 🚀 Key Features

- **✅ Online Test Engine UI:** Professional interface for Verbal, Non-Verbal, and Academic tests.
- **🏆 Grand Test Layout:** A simulated exam environment designed for optimal user experience.
- **⚖️ Health Tools:** 
  - **BMI Calculator:** Check if you meet the height/weight standards.
  - **Age Calculator:** Verify your eligibility for military courses (e.g., PMA Long Course).
- **📱 Responsive UI:** Fully optimized for mobile and desktop with a premium military-themed design.
- **🔥 Modern Tech Stack:** Built using HTML5, Vanilla CSS, and JavaScript.
